# Quick Start

1. Extract the ZIP first.
2. Run `START-WaibonDevShield.cmd`.
3. Paste a project folder path.
4. Choose scan mode:
   - Quick Scan: fast preliminary pre-open review. It may show more review warnings because it checks early signals first.
   - Deep Scan: slower behavior/context review. It may reduce false positives by reading more context and shows elapsed/ETA-style progress.
5. After scan completion, the HTML report opens automatically.

Safety mode: report only. No delete, modify, quarantine, or target-file execution.
