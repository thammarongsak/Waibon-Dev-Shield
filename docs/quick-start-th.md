# วิธีใช้เร็ว

1. แตก ZIP ก่อน
2. เปิด `START-WaibonDevShield.cmd`
3. วาง path โฟลเดอร์โปรเจกต์
4. เลือกโหมดสแกน:
   - Quick Scan: ตรวจเร็วแบบเบื้องต้นก่อนเปิดโปรเจกต์ อาจเตือนมากกว่า เพราะดูสัญญาณเร็วเป็นหลัก
   - Deep Scan: ตรวจช้ากว่า แต่อ่านบริบทและพฤติกรรมมากกว่า อาจลด false positive ได้ดีขึ้น และมีเวลา/ETA ให้ดูระหว่างสแกน
5. หลังสแกนจบ ระบบจะเปิดรายงาน HTML ให้อัตโนมัติ

โหมดปลอดภัย: รายงานอย่างเดียว ไม่ลบ ไม่แก้ ไม่กักไฟล์ และไม่รันไฟล์เป้าหมาย
