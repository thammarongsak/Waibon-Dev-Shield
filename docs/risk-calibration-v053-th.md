# การปรับน้ำหนัก v0.6.0

Waibon Dev Shield แยก Critical กับ Warning ชัดขึ้น และเพิ่ม Strict Warning Gate เข้ามาอีกชั้น ข้อความที่ตรวจพบและระดับ rule เป็นเพียงหลักฐานดิบ ไม่ใช่คำตัดสิน สีแดงจะขึ้นยากขึ้น ต้องมีหลักฐานหนักจริง ส่วนสัญญาณที่อยู่ใน docs/test/example/placeholder/detector rule จะถูกลดระดับลงเป็น Review/Info มากขึ้น
