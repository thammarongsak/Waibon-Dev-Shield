# Root Cause Grouping

Waibon Dev Shield v0.6.0 แยก “สัญญาณดิบ” ออกจาก “ประเด็นที่ควรตรวจจริง”

- Raw signals คือจุดที่ตรวจพบจากข้อความหรือ rule รายรายการ
- Unique review issues คือการรวมสัญญาณซ้ำที่มาจากไฟล์ rule และเจตนาเดียวกัน
- Critical root causes และ Warning issues คือกลุ่มประเด็นที่ควรรีวิว ไม่ใช่การกล่าวหาว่าไฟล์เป็นมัลแวร์

แนวนี้ช่วยลดการตีความเกินจริงเมื่อไฟล์เดียวหรือสคริปต์เดียวทำให้เกิดสัญญาณซ้ำจำนวนมาก
