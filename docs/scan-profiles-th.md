# Scan Profiles

v0.6.0 เพิ่มโหมดสแกนเฉพาะทาง:

1. Quick Scan - ตรวจเร็วก่อนเปิดโปรเจกต์
2. Smart Deep Scan - โหมดละเอียดแบบสมดุล แนะนำให้ใช้
3. Full Deep Scan - กว้างที่สุดและช้าที่สุด
4. Secret & Token Scan - เน้น token, key, .env
5. Supply-chain Scan - เน้น package hooks, CI/CD, install surface
6. AI Agent / MCP Scan - เน้นไฟล์คำสั่ง agent และ MCP/tool config
